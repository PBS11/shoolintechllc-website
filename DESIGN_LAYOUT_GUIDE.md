## 🎨 Color Scheme

**Primary Colors:**
- Primary Blue: `#0066cc` - Main brand color
- Primary Dark: `#0052a3` - Button hover state
- Accent Orange: `#ff6b35` - Highlights & accents

**Backgrounds:**
- White: `#ffffff` - Section backgrounds
- Light Gray: `#f8f9fa` - Subtle backgrounds
- Light Blue: `#f0f4f8` - Gradient backgrounds

## 📐 Responsive Breakpoints

| Device | Layout |
|--------|--------|
| Desktop (1200px+) | Multi-column grids, side-by-side layouts |
| Tablet (768px-1199px) | 2-column grids, adjusted spacing |
| Mobile (<768px) | Single column, stacked elements |

---

## ✨ Interactive Elements

### **Hover Effects**
- Cards lift up with shadow enhancement
- Buttons scale and change opacity
- Links get underlines or color changes

### **Animations**
- Logo floats gently up/down (3s loop)
- Hero content slides in on load
- Visual elements rotate continuously

### **Transitions**
- All hover effects use 0.3s smooth transitions
- Cubic-bezier timing for natural motion
- Scale transforms for interactive feedback

| Element | Size | Weight |

| Hero Title | 3.5rem | 800 Bold |

| Card Heading (h3) | 1.3rem | 700 Bold |
| Body Text | 1rem | 400 Normal |
| Small Text | 0.9rem | 500 Medium |

---

## 🎯 Key Design Principles

1. **Visual Hierarchy** - Important content is larger and more prominent
2. **Whitespace** - Generous spacing for breathing room
3. **Consistency** - Same design patterns throughout
4. **Modern Aesthetics** - Gradients, shadows, rounded corners
5. **Performance** - CSS animations for smooth experience
6. **Accessibility** - Good contrast, readable fonts
7. **Responsive** - Works perfectly on all screen sizes
8. **Professional** - Corporate but modern appearance

---

## 📊 Content Sections (Top to Bottom)
